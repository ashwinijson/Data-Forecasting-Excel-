# YouTube Channel Forecasting Project

## 1. Overview
An Excel-based forecasting project to predict YouTube channel views using three methods —
Forecast Formula, Trendline Graph, and Forecast Sheet — showcasing different forecasting
techniques and accuracy comparisons

## 2. Data Structure
Tool: Microsoft Excel
Data Range: 01-Jun-2022 to 14-Jan-2023 (Actuals)
Forecast Period: 15-Jan-2023 to 21-Jan-2023
Metric: View Count per Day/Week

## 3. Execution Summary
Structured historical YouTube data and applied three forecasting methods. Visualized predicted
trends through charts and forecast sheets to identify upcoming engagement patterns.

## 4. Insights & Deep Dive
Consistent upward trend in forecasted views.
Forecast Sheet provided most clear visual representation.
Trendline captured gradual growth pattern effectively.

## 5. Recommendations
Use Forecast Sheet for presentations.
Extend forecast to track longer-term engagement trends.
Compare results across methods for accuracy improvement.

## Tools & Skills:
Excel • Forecast.ETS • Trend Analysis • Visualization • Predictive Analytics
